## 🐛 Обнаруженные проблемы

### Критические:
1. **Deployment использует неправильный скрипт** - запускается `start_with_graphql.py` вместо `start_api_only.py`
2. **GraphQL API не работает** - сервер зависает при старте
3. **Telegram bot конфликты** - множественные экземпляры пытаются подключиться
4. **Health check failing** - приложение не проходит проверку здоровья

### Средние:
1. **WebSocket недоступен** - dashboard не может получать real-time обновления
2. **Долгий рестарт** - приложение долго перезапускается
